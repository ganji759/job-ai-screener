# Umurava AI HR

AI-powered talent screening platform.

## Structure

- `backend/` — Node.js 20 + TypeScript + Fastify API + BullMQ worker + Python AI service
- `frontend/` — Next.js 14 + Redux Toolkit + Tailwind CSS recruiter dashboard

## Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```

See `backend/README.md` and `frontend/README.md` for full setup instructions.
